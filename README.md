# MSL Motion Analyzer

[![GitHub Pages](https://img.shields.io/badge/deployed-GitHub%20Pages-blue?style=for-the-badge&logo=github)](https://pfmkawachi.github.io/msl-ma/)
[![License: MIT](https://img.shields.io/badge/License-MIT-emerald?style=for-the-badge)](LICENSE)

スマートフォン、タブレット、PCのあらゆるデバイスでの動作分析を可能にする、高精度なWebベースのモーションアナライザーです。

サーバーへの動画アップロードは一切不要。完全ローカル処理（ブラウザ内完結）のため、大容量の動画でも一瞬で読み込め、プライバシーや通信量を気にする必要はありません。

---

## ✨ 主な機能

- **スマートデバイス最適化**: アクセスした端末（iOS / Android / PC）を自動で判別し、最適なUI（モバイル版 / デスクトップ版）へ自動振り分け。
- **高精度コマ送り**: 24 / 30 / 60 FPSに対応した「+1F / -1F」「+5F / -5F」のフレームシーク。
- **自由なズーム & パン（位置移動）**: 
  - スマホ・タブレット：直感的な2本指ピンチ・パン操作
  - PC：マウスホイールでのズーム ＆ ドラッグでの位置移動
- **A-B間ループ再生**: 指定した2点間（A点・B点）を自動で繰り返しループ再生。フォームの反復チェックに最適。
- **左右反転（ミラー機能）**: ワンタップで動画を左右反転。利き手などに合わせた分析が可能。
- **キーボードショートカット (PC/タブレット)**: `Space`で再生/停止、`矢印キー`で正確なコマ送り操作が可能。
- **スタンドアロン起動対応**: ホーム画面に追加することで、ブラウザのツールバーを非表示にし、ネイティブアプリのように全画面で起動可能。

## 📂 ファイル構成

本リポジトリはサーバー（バックエンド）を必要としない、フロントエンド完結型の静的サイトです。

```text
├── index.html          # デバイス自動判別ゲートウェイ
├── mobile.html         # スマートフォン専用 UI
├── desktop.html        # PC & タブレット専用 UI
├── site.webmanifest    # Android用PWA設定ファイル
└── img/                # 📁 ロゴやアイコンなど
```
